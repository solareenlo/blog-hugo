# [Hugo](https://github.com/gohugoio/hugo)とは
静的なhtmlを生成する事ができる静的サイトジェネレータの1つ.
Go言語で書かれてる.
記事を書くだけならGo言語を知らなくても良い.
記事はMarkdownで書ける.

## .mdを削除した時に.htmlも同様に削除するオプション
```
hugo server -D --cleanDestinationDir
```
