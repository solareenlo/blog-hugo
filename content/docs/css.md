```css
<div class="contaiiner"> /* 固定枠 */
  <div class="row"> /* row枠 */
    <div class="col-xs-12"></div> /* スマホ枠 */
    <div class="col-sm-12"></div> /* タブレット枠 */
    <div class="col-md-12"></div> /* PC枠 */
    <div class="col-lg-12"></div> /* PC大枠 */
  </div>
</div>
```
